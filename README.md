# Instalacion en Termux

# 📲Termux
Para nuevos desde cero en termux.apk:
```bash
> termux-setup-storage
> termux-change-repo
> apt update -y && apt upgrade -y
> pkg install git
> pkg install nodejs
> pkg install libwebp -y
> pkg install ffmpeg -y
> pkg install imagemagick -y
> pkg install yarn

```
